Social-media

Primer proyecto fullstack que desarrolle, hace mucho estaba buscando un proyecto que cumpliera con las necesidades de utilizar todas las tecnologias necesarias para un desarrollo de frontend y backend con llamadas a bases de datos en la nube, entonces investigando al respecto me encontre con la posibilidad de realizar una red social con funcionalidad basica pero necesaria en cualquiera de este tipo. El usuario puede crear una nueva cuenta con un formulario que realiza las respectivas validaciones, puede iniciar sesion, realizar un posteo y comentar en posteos de otros usuarios, la interfaz se actualiza en cada acción que se realiza (Utilizando React-Redux) , ademas el usuario tiene un perfil personal en el cual puede eliminar los posteos que realizó!

La complejidad mas alta que tuve al realizar este proyecto fue la de mantener la sesion del usuario al refrescar la pagina. En principio lo realicé generando una cookie al inicio de sesión del usuario, luego me encontre en la posibilidad de utilizar un contexto para mantener la sesión en el estado de la aplicacion constantemente pero no llegue a implementarlo, si bien la aplicacion funciona de manera correcta uno siempre quiere que el codigo sea lo mas mantenible y escalable posible.

Leaf-store

Utilice Material-UI para el estilo de los componentes y para el tema verde que se puede observar, este proyecto fue de una complejidad baja.
